# Lab 3: HTML only

## Chiikawa Adventure

Este proyecto consiste en una historia interactiva desarrollada únicamente con HTML.  
Cada decisión lleva a diferentes rutas, incluyendo finales buenos, malos y un final secreto.

La navegación utiliza:

- Múltiples páginas HTML
- Paths relativos y absolutos
- Estructura semántica (header, main, article, section, aside, nav, figure, etc.)
- Breadcrumbs en todas las páginas
- Barras de progreso
- Uso de `<time>` en eventos narrativos
